# Changelog

All notable changes to SignAsili will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-05-05

### Added

- Complete KSL education platform for 600,000+ deaf Kenyans
- FastAPI backend with 50+ API endpoints
- Next.js 14 frontend with React 18.3
- PostgreSQL database with 30+ tables
- JWT authentication with MFA support
- Role-based access control (learner, teacher, parent, admin, EARC officer)
- Interactive 6-phase lesson structure
- AI-powered sign recognition using MediaPipe Holistic
- 3D IMARA avatar with lip-sync
- Teacher dashboard with SLO mastery heatmap
- Bridge Programme for hearing parents
- Parent-child co-op challenges
- KSL card generation and verification
- Community circles for deaf learners
- Offline-first mobile architecture
- M-PESA payment integration
- NEMIS school system sync
- Docker Compose for development
- Kubernetes manifests for production
- GitHub Actions CI/CD pipeline
- Comprehensive documentation

### Security

- JWT tokens with 15-minute expiry
- HTTP-only refresh cookies
- Password strength validation
- Rate limiting on auth endpoints
- Audit logging
- Input validation with Pydantic

### Technical

- 75+ database tables
- 50+ API endpoints
- 21,105 training videos from 6 Kenyan schools
- 100+ KSL signs in database
- mT5 translation model
- LSTM sign detection model
- Temporal CNN lip sync model
